Directions:

1) Present on this page, the blockchain parameters that a customer can select

2) Generate (if not available) wallet and create token

3) Generate json for chain validation request (Chain entity, parameters in properties)

4) AJAX: call /rest/iaas/validate_chain_creation_request, which returns the $$$ if validated

5) If validated, pay $PDX on the service chain via /rest/iaas/forward_iaas_payment

6) Inject payment TXID to the above chain validation request's properties

7) AJAX: call /rest/iaas/orchestrate_chain_creation to create the chain.

Notes:

1) The customer can check its status later with the chain name (chain id)

2) Iaasservice.orchestrateChainCreation submits the creation to ChainCreationService.
